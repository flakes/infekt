iNFekt - FAQ

The rendered mode is slow.

Please try to disable "Hardware (GPU) acceleration" in the preferences' Advanced section.

How do I open the menu?

Simply press the Alt key on your keyboard. You can also permanently show it using the menu button from iNFekt's toolbar.

I get an error message "Side-by-side configuration is incorrect" or "MSVCR142.DLL is missing" when I try to start the application.

You need to download and install the Visual C++ Redistributable Package from Microsoft. Alternatively, you can get the iNFekt setup file which will download and install the required files automatically.

  • Up to and including version 1.0.1 the Visual C++ 2015 runtime was required: Microsoft Visual C++ Redistributable for Visual Studio 2015

  • Up to and including version 0.9.5 the Visual C++ 2012 runtime was required: Microsoft Visual C++ Redistributable for Visual Studio 2012

  • Up to and including version 0.8.5 the Visual C++ 2010 runtime was required: Microsoft Visual C++ 2010 SP1 Redistributable Package (x86) or Microsoft Visual C++ 2010 SP1 Redistributable Package (x64)

Where can I get a Windows XP/7/8 compatible version?

You can use iNFekt 1.0.1.

Where can I get a Windows 2000 compatible version?

You can use iNFekt 0.7.9 (32-bit).

Where can I get a Windows XP 64-bit compatible version?

You can use iNFekt 0.8.5 or any 32-bit version.
